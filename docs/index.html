Follow the steps below to get started.

1. Download the app

Tap the button below to download the file. It will land in your Downloads folder.

2. Allow installation

When you open the file, Android will ask if you want to allow installation from this source. Tap Allow.

If it asks you to enable unknown sources first:

  1. Tap Settings
  2. Toggle on Allow from this source
  3. Head back and tap Install

You may also see a secondary prompt from Play Protect recommending an app scan. If you do:

  1. Tap the More Options dropdown menu
  2. Select Install without scanning

3. Open the app and go

Once installed, open the app.
